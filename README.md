# Indicador Estado Monotributo
### Sistema simple para cargar facturas, donde muestra el estado actual (categoria y saldo hasta proxima categoria)


#### Backend en nodeJS
┌────────────────────────────────────────────┬─────────────────────────────────────────────────────────────────────────────────────────┐                                                                                                            
│                  Endpoint                  │                                       Descripción                                       │                                                                                                            
├────────────────────────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────┤                                                                                                            
│ GET /api/escalas                           │ Devuelve letra_escala, ingresos_brutos_anuales_max, cuota_total_locaciones_y_servicios  │                                                                                                            
├────────────────────────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────┤                                                                                                            
│ POST /api/escalas/sync                     │ Fuerza resincronización con AFIP                                                        │                                                                                                            
├────────────────────────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────┤                                                                                                            
│ POST /api/facturas                         │ Body: { tipo, fecha, destinatario, monto }                                              │                                                                                                            
├────────────────────────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────┤                                                                                                            
│ GET /api/facturas?modo=anual&anio=2024     │ 01/01/2024 – 31/12/2024                                                                 │                                                                                                            
├────────────────────────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────┤                                                                                                            
│ GET /api/facturas?modo=semestral&anio=2024 │ 01/07/2024 – 30/06/2025                                                                 │                                                                                                            
└────────────────────────────────────────────┴─────────────────────────────────────────────────────────────────────────────────────────┘

Al arrancar el servidor sincroniza las escalas de AFIP automáticamente.
